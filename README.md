# Simple To-Do List Application

This is a beginner-level To-Do List built using HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Mark tasks as completed by clicking on them
- Delete tasks by double-clicking

## Objective
The purpose of this project is to practice DOM manipulation and event handling using JavaScript.

## How to Use
1. Open `index.html` in your browser
2. Type a task in the input box and click "Add Task"
3. Click on a task to mark it as completed
4. Double-click on a task to delete it
